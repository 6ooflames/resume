# 3D Showcase Portfolio

This directory contains 3D assets, project files, and exports showcasing my work bridging systems architecture with creative graphics pipelines.

## Accessing the Showcase

Please view the interactive gallery at [https://6ooflames.github.io/resume/showcase/](https://6ooflames.github.io/resume/showcase/) (or locally by opening `index.html` in this folder).

## Directory Structure
- `index.html`: Interactive gallery showcasing the assets below.
- `assets/`: (To be populated) Contains `.glb`, `.webp` thumbnails, and optimized web exports.
- `bremen.blend`: Sample blender project file.

## Managing Large Files

Due to GitHub's file size limitations (100MB per file), full-resolution `.blend` files for projects like the 774MB Tesla Model and 1.7GB Vault 111 scene should be:
1. Tracked using [Git LFS](https://git-lfs.com/)
2. Hosted externally
3. Uploaded as a [GitHub Release](https://docs.github.com/en/repositories/releasing-projects-on-github)

Currently included is `bremen.blend` (~25MB) as an accessible showcase item.
